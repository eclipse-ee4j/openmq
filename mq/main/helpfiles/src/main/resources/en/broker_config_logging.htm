Logging

Use this tab to configure logging for a broker instance.

Log Level Specifies the level of logging output that can be written to any output channel. The categories, in decreasing order of severity, are ERROR, WARNING, INFO. Use the drop list to select a category. When you select a category, the logger writes out data of the specified category and all higher categories as well. For example, if you select INFO, you will get data in all three categories. If you select ERROR, you will only get ERROR messages. The default is INFO.

Log Rollover Size Specifies the size of the log file at which output rolls over to a new log file. Use the drop list to select the units: bytes, kilobytes, or megabytes. Select the Unlimited radio button to specifiy no rollover based on log file size. The default selection is Unlimited.

Log Rollover Interval Specifies the age, in seconds, of the log file when output rolls over to a new log file. Select the Unlimited radio button to specifiy no rollover based on age of file. Default is 604800.
