Messaging

Use this tab to configure messaging capacity and message size for this instance of the broker.

Max Number of Messages in System Specifies maximum number of messages in both memory and disk (due to swapping). Additional messages will be rejected. Default is Unlimited.

Max Total Message Bytes in System Specifies maximum total size (in bytes, Kbytes, or Mbytes) of messages in both memory and disk (due to swapping). Additional messages will be rejected. Default is Unlimited. Use the drop list to specify the units used to calculate the size: bytes, kilobytes, or megabytes.

Max Message Size Specifies maximum allowed size (in bytes, Kbytes, or Mbytes) of a message body. Any message larger than this will be rejected. Default is 70 megabytes. Use the drop list to specify the units used to calculate the size: bytes, kilobytes, or megabytes.
