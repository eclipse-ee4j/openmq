3.0 Connection Handling

Use the Connection Handling tab to reconfigure basic information about the connections that will be created by this factory.

Connection Type Use the drop list to select the connection type.

Broker Host Name Specify the name of the machine hosting the broker to which the client will connect.

Broker Host Port Specify the number of the port-mapper port. The client uses the information provided by the port-mapper to connect to the desired service, either jms or ssljms.

Broker Service Port Specify the number of the port to use to make a direct connection to the jms or ssljms service. Use Broker Service Port to bypass the port-mapper. This allows clients to connect through firewalls using a single known port.

HTTP URL Specify the URL that will be used to connect to the Message Service. By default this is http://localhost/imq/tunnel.
