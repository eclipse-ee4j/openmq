Broker Destination Properties

This dialog allows you to view the current state of a broker destination and to change the properties that have been specified for that destination.

Use the Basic tab to get information about the destinations current state and to change selected properties. Use the Durable Subscriptions tab to get information about durable subscription that have been created on this broker.
